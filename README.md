# URN-CAB
This is about the benchmark, source code, solution certificates of our union-reduced neighborhood heuristic algorithm for the cyclic antibandwidth problem (CAB, as a famous NP-hard problem).

------------------------------------------------------------------------
Benchmark:

Totally 295 instances in 12 diffirent sets, including 5 sets with theoretically optimial solutions, 3 sets with conjectured optimal solutions and 4 sets with unknown optimal solutions.

- The first 8 sets are employed by Bansal and Srivastav[1].

- The other sets are introduced by  Lozano et al. [2].
  
------------------------------------------------------------------------
Source Code:
The manuscript is under reivew. The source code of the proposed algorithm will be uploaded upon the accptance of the paper.

------------------------------------------------------------------------
Solution certicates:

The solution for each instance labeled node from 0 not 1.
